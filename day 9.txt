Write a function that finds odd number in a given number and checks prime number digit in those odd number 
	if no prime number exits then 
		print 0
	if one prime number exits then 
		print prime_number
	if two prime number exits then
		print first prime number only
	else if more than three prime number exits
		print last prime number only
Note : take interger number from user

Solution :
def prime(num):
    if num > 1:
        for i in range(2, int(num/2)+1):
            if (num % i) == 0:
                return False
                break
        else:
            return True
    else:
        return False
